## Nest Template

Template of a nest production-ready application with Prettier, ESlint, and Pre-commit

## Installation

```bash
$ npm install
```

```bash
# Install dependencies
$ npm install

# Generate primsa database
$ npx prisma generate

# Push databases tables
$ npx prisma db push
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start-dev

# production mode
$ npm run start-prod
```
