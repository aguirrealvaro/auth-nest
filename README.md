## auth-nest

Backend api with auth made with Nest, Prisma and Postgres

## Installation

```bash
$ npm install
```

```bash
# Install dependencies
$ npm install

# Generate primsa database
$ npx prisma generate

# Push databases tables
$ npx prisma db push
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start-dev

# production mode
$ npm run start-prod
```
